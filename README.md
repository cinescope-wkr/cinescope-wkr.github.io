# Academic Homepage Template

This repo now ships with a single-page academic/personal homepage template
inspired by the structure of `nmwsharp.com`.

## Customize

- Edit `index.html` to replace the placeholder biography, project entries, news,
  social links, and CV URL.
- Edit `css/style.css` to adjust fonts, colors, spacing, and card styling.
- Replace the `[YN]` monogram block with a headshot image if you want a portrait.

## Current footprint

- `index.html`
- `css/style.css`
- `favicon.svg`
