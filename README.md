# Personal Website Template

This repo was reduced from a large multi-page academic site to a minimal static
template for a personal homepage.

## Customize

- Edit `index.html` to replace the placeholder copy, links, and sections.
- Edit `css/style.css` to change the palette, typography, spacing, and layout.
- Add a `CNAME` file later if you want to attach a custom GitHub Pages domain.

## Current footprint

- `index.html`
- `css/style.css`
- `favicon.svg`
